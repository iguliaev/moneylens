# Visual Identity Mockup Design — Quiet Ledger

**Date:** 2026-05-30  
**Scope:** Visual identity mockups for MoneyLens  
**Status:** Draft for review

## Goal

Create a calm, premium mockup direction for MoneyLens that feels trustworthy, polished, and easier to scan than the current generic finance UI.

## Chosen Direction

**Quiet Ledger**

This direction uses:

- a warm paper-like background
- soft, rounded cards with light borders
- a restrained cobalt blue primary accent
- a simple **icon + wordmark** logo lockup
- a slightly editorial layout rhythm

The goal is not to look flashy. The goal is to feel considered and quietly premium.

## Visual Principles

1. **Clarity first** — numbers and cashflow signals should be instantly readable.
2. **Quiet surfaces** — backgrounds stay soft and warm instead of stark white.
3. **One strong accent** — blue carries navigation, emphasis, and brand recognition.
4. **Financial signal colors** — green and red are reserved for income and expense states.
5. **Crafted logo** — the lens mark should feel like a brand asset, not text in SVG.

## Mockup Components

### 1. Header / Brand

- Use an icon + wordmark treatment.
- The icon is a simple lens/circle mark with a highlight.
- The wordmark should read clearly at small sizes.

### 2. Dashboard Hero

- Large summary figure.
- Short supporting line explaining the month state.
- Three compact metrics beneath it.
- Card-based chart block using the same blue accent.

### 3. Transaction / Category Table

- Soft table rows with clear spacing.
- Minimal chrome.
- Positive values in green, negative values in red.

### 4. Auth Screen

- Reuse the same visual language on login/register pages.
- Same paper surface, same rounded inputs, same blue button.
- The auth page should feel like part of the same system, not a separate template.

### 5. Visual System Panel

- Show the palette explicitly.
- Keep the accent set small and disciplined.

## Color Direction

- **Primary:** cobalt blue
- **Surface:** warm paper / ivory
- **Positive:** green
- **Negative:** red
- **Text:** deep slate

## Typography Direction

- Use **DM Sans** for the wordmark and UI text.
- Prefer a clean, modern, slightly soft tone.
- Avoid overly geometric or overly corporate typefaces.

## What the Mockup Should Communicate

- MoneyLens is calm, not noisy.
- The product is practical, but still designed.
- The identity can extend cleanly into charts, auth pages, and small icons.

## Non-Goals

- No maximalist visual effects.
- No neon-heavy palette.
- No large motion system.
- No mobile layout redesign in this phase.
- No final production implementation decisions yet.

## Success Criteria

The mockup is successful if it:

- feels premium without becoming busy
- works as a believable basis for the app shell and auth screens
- makes the brand lockup feel native to the product
- stays visually consistent across light and dark references
